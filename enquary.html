🔥 Firebase Setup Guide
Follow these steps to connect your app to Firebase Realtime Database and host on GitHub Pages.
1
Go to console.firebase.google.com → Create a new project
2
In your project: Build → Realtime Database → Create database → Start in test mode
3
Go to Project Settings → Your Apps → Add Web App → Copy the firebaseConfig object
4
Open this HTML file → find FIREBASE_CONFIG at the bottom → paste your config values
5
For GitHub Pages: Create a GitHub repo → upload this HTML file as index.html → Settings → Pages → Deploy from main branch
6
In Firebase Console → Realtime Database → Rules → set read/write rules for your domain
